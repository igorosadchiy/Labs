Create a page which will have next items:

1. Title of the city you would like to visit - use <h1> tag;

2. Description of that city - use <p> tag;

3. Highlight important words in the description using <b> tag;

4. Unordered list of other cities you would like to use - use <ul>;

5. Add image of the city - use <img> tag and src attribute;

6. Add <iframe> of the city from wiki page - use <iframe> tag and src, width, height attributes;

7. Add <title>;

Создайте страницу, на которой будут следующие элементы:

1. Название города, которое вы хотели бы посетить - используйте тег <h1>;

2. Описание этого города - используйте тег <p>;

3. Выделите важные слова в описании, используя тег <b>;

4. Неупорядоченный список других городов, которые вы хотели бы использовать - используйте <ul>;

5. Добавить изображение города - используйте тег <img> и атрибут src;

6. Добавьте <iframe> города из wiki-страницы - используйте тег <iframe> и атрибуты src, width, height;

7. Добавьте <title>;
